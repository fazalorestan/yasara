# Apply YaSara v4.18 One Command Launcher

Extract into:

```text
D:\yasara_clean
```

Run:

```cmd
cd /d D:\yasara_clean
python scripts\apply_v4_18_launcher_patch.py
python yasara.py patch
python yasara.py start
```

After this, dashboard should open automatically:

```text
http://127.0.0.1:5173
```

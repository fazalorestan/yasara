# YaSara v4.27 — Extension Host & Plugin Sandbox

## Goal
Provide an Extension Host layer so future plugins can run independently while preserving Core stability.

## Scope
- Infrastructure only
- Report-only resource quotas
- No trading feature
- No dashboard change
- No existing API rewrite

## Adds
- Extension Host
- Plugin Sandbox
- Resource Quota Reporter
- Plugin Health Monitor
- Startup Profiler
- Dependency Graph
- Safe Shutdown Contract
- Recovery Manager

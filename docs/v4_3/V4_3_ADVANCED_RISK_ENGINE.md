# YaSara v4.3 — Advanced Risk Engine Foundation

## Goal
Add a modular risk engine for Signal, Backtest, Paper Trading and future Personal-only Execution.

## Includes
- Dynamic Position Size
- Dynamic Leverage Guard
- Risk Per Trade
- Daily / Weekly / Monthly Risk
- Portfolio Risk
- Max Drawdown
- Kelly Criterion
- Circuit Breaker
- Kill Switch
- Recovery Mode

## Safety
No real execution. This phase only calculates and gates risk.

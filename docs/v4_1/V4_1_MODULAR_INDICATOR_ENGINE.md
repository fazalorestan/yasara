# YaSara v4.1 — Modular Indicator Engine Foundation

## Goals
- Add independent modular indicators.
- Keep all current APIs intact.
- Provide a unified indicator output format.
- Feed future AI, Signal, Risk, Backtest and Market Context layers.

## Indicators included
EMA, SMA, RSI, MACD, ATR, ADX, SuperTrend, VWAP, Bollinger, OBV, MFI, CCI, Stochastic, Donchian, Keltner, Parabolic SAR.

## Safety
Analysis only. No execution engine. Live trading remains disabled.

# YaSara v5.0-alpha.16 — Exchange Connector Framework

## Goal
Create a plugin-ready exchange connector framework before adding real exchange API integrations.

## Scope
- Connector framework only
- No real exchange connection
- No broker execution
- No auto trading
- No dashboard redesign
- Backward compatible

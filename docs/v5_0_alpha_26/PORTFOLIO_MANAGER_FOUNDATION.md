# YaSara v5.0-alpha.26 — Portfolio Manager Foundation

## Goal
Create portfolio contracts and aggregation services before scanner, alerts and backtest integration.

## Scope
- Portfolio contracts only
- No real broker connection
- No real execution
- No auto trading
- No dashboard redesign
- Backward compatible

## Adds
- Portfolio Snapshot
- Holding Contract
- Allocation Summary
- Equity Curve Contract
- PnL Summary
- Portfolio Exposure
- Portfolio Risk Link
- Readiness Gate

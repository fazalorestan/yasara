# YaSara v5.0-alpha.3 — Indicator Sandbox & Validation Foundation

## Goal
Create a safe sandbox and validation foundation for future indicator plugins.

## Scope
- Validation and safety contracts only
- No live trading
- No new market data connection
- No dashboard redesign
- Backward compatible

## Adds
- Indicator Sandbox Contract
- Manifest Validator
- Runtime Output Validator
- Security Policy Validator
- Plugin Isolation Check
- Install Safety Gate

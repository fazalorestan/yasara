# YaSara v5.0-alpha.24 — Runtime API Smoke Test Integration

## Goal
Add a runtime API smoke-test foundation so missing routers and broken endpoints can be detected automatically.

## Scope
- API smoke testing infrastructure only
- No new trading feature
- No real execution
- No dashboard change
- Backward compatible

## Adds
- Runtime Endpoint Catalog
- HTTP Smoke Contract
- Router Registration Audit
- Endpoint Status Validator
- Runtime API Smoke Plan
- Readiness Gate

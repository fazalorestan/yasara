# YaSara v5.0-alpha.4 — Indicator Lifecycle & State Manager

## Goal
Create a safe lifecycle and state manager foundation for indicator plugins.

## Scope
- Lifecycle/state contracts only
- No live trading
- No new market data connection
- No dashboard redesign
- Backward compatible

## Adds
- Indicator Lifecycle State
- Discovered / Validated / Installed / Enabled / Disabled
- Lifecycle Transition Rules
- State Snapshot
- Safe Rollback Contract
- Lifecycle Audit Event

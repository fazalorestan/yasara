# YaSara Professional v1.1 — Phase 2: Exchange Connectivity Layer

This phase adds a production-safe read-only exchange connectivity layer.

## Capabilities

- Exchange endpoint profiles
- Read-only REST request builder
- Safety guard
- Health service
- Failover router
- Rate limit profiles
- Connectivity router
- Diagnostics API

## Safety

No live trading. Signed/private requests are blocked by default.

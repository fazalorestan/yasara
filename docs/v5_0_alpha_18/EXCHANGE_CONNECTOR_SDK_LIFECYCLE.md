# YaSara v5.0-alpha.18 — Exchange Connector SDK & Plugin Lifecycle

## Goal
Create a stable SDK and lifecycle foundation for future real exchange connector plugins.

## Scope
- SDK foundation only
- No real exchange connection
- No broker execution
- No auto trading
- No dashboard redesign
- Backward compatible

## Adds
- Base Exchange Connector SDK
- Connector Lifecycle States
- Connector Manager
- Connector Health Watchdog
- Capability Negotiation
- Connector Sandbox Contract
- Exchange Event Contract
- Readiness Gate
